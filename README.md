# CS AI - 人工智能

人工智能知识库，包含机器学习、深度学习、自然语言处理、计算机视觉等AI核心技术。

## 目录结构

- [machine-learning](./machine-learning) - 机器学习
- [deep-learning](./deep-learning) - 深度学习
- [natural-language-processing](./natural-language-processing) - 自然语言处理
- [computer-vision](./computer-vision) - 计算机视觉

## 内容概述

### 机器学习
监督学习、无监督学习、强化学习、特征工程、模型评估、算法原理等。

### 深度学习
神经网络、CNN、RNN、Transformer、GAN、模型优化、框架使用等。

### 自然语言处理
文本预处理、词向量、序列标注、机器翻译、情感分析、大语言模型等。

### 计算机视觉
图像处理、目标检测、图像分割、图像生成、视频分析、视觉Transformer等。
